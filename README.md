# Password Utility Tool 

A Python-based password utility that evolves from a simple checker into a full-featured CLI application. Built to practice core programming concepts and demonstrate progressive development.

---

## Features
- Password strength checker (rule-based validation)
- Secure password generator with customizable options
- Save and view password history (JSON storage)
- Interactive command-line menu system

---

## Project Stages
- v1: Basic password checker using conditionals  
- v2: Strength rules system  
- v3: Password generator  
- v4: File storage (JSON history)  
- v5: Full CLI application  

---

##  Technologies
- Python 3
- JSON for data storage
- Random & string modules

---

##  Future Improvements
- GUI version (Tkinter)
- Password encryption
- Web-based version using Flask

---

##  Purpose
This project demonstrates progression in Python fundamentals, file handling, and building structured CLI applications.
